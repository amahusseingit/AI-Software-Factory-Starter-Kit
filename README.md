# Agentic SDLC Startup Kit V5

Enterprise governance execution kit for running a full AI-driven SDLC from BRD to completion.

## What V5 adds
- approval gates and sign-off templates
- RACI matrix template
- sprint and release templates
- KPI dashboard templates for productivity and quality
- Jira-ready issue templates and workflow mapping
- expanded governance artifacts on top of V4 execution kit
- full timesheet and run-log enforcement across all roles

## Primary entry point
1. Put the BRD in `specs/BRD.md`
2. Start with `prompts/START-PROJECT.md`
