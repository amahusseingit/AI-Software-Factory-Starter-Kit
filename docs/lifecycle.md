# Lifecycle

## DEFINE
Goal: establish scope, workflows, roles, rules, validations, integrations, and constraints.

## PLAN
Goal: convert scope into verifiable vertical slices.

## BUILD
Goal: implement behavior, not shells.

## VERIFY
Goal: validate correctness through tests and evidence.

## REVIEW
Goal: block incomplete, unsafe, or low-quality delivery.

## SHIP
Goal: make release practical and supportable.
