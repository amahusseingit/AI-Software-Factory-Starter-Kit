# V7.5 Usage Guide

## When to use NFR packs
Use them whenever the project needs explicit quality attributes such as security, performance, availability, accessibility, or observability.

## When to use ADR packs
Use them whenever architecture choices materially affect implementation structure, runtime behavior, or operational burden.

## When to use DevOps packs
Use them before release or whenever environment, pipeline, deployment, migration, rollback, or runtime readiness must be defined.
