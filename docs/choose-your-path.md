# Choose Your Path

## I want to start a new system
Use:
- `specs/<your-project>.md`
- `commands/START-PROJECT.md`

## My requirements are weak or incomplete
Use:
- `commands/SPEC.md`
- spec-quality analysis skill
- project spec and UI templates

## I have a half-complete AI-generated system
Use:
- `commands/RESCUE-PROJECT.md`
- rescue guidance and salvage-vs-rebuild templates

## I want review and release readiness
Use:
- `commands/REVIEW.md`
- `commands/SHIP.md`
- review scorecards and readiness templates
