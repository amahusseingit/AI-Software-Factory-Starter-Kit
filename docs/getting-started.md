# Getting Started

## Quick start
1. Add your project spec under `specs/`.
2. Read `AGENTS.md`.
3. Start with `commands/START-PROJECT.md`.
4. Continue through `SPEC`, `PLAN`, `BUILD`, `TEST`, `REVIEW`, and `SHIP`.
5. Use `commands/RESCUE-PROJECT.md` if the system already exists but is incomplete.

## What to prepare
- one authoritative project spec
- your preferred AI tool
- agreement on review and evidence expectations
