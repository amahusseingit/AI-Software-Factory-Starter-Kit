# Code of Conduct

## Our pledge
We want this project to be a respectful, constructive, and inclusive place for collaboration.

## Expected behavior
- Be respectful and professional.
- Give actionable feedback.
- Focus on improving the project.
- Assume good intent, but be clear and direct.

## Unacceptable behavior
- Harassment or abusive language
- Personal attacks
- Discrimination
- Repeated bad-faith disruption

## Enforcement
Project maintainers may remove or reject contributions that violate this code of conduct.
